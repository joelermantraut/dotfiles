# dotfiles
All my config dotfiles.

This repository has all the config files that I have in my Arch Linux instalation. 

## It includes:
- The default windows manager is i3. It's really well configured.
- There is another windows manager installed, Openbox, which is a floating WM.
- The terminal is Alacritty.
- The text editors are Neovim and Sublime-text.
- The main aplicattion launcher is Rofi. I also use dmenu_extended for some purposes.
- The default web browser is Brave Browser.
- The default file manager is Pcmanfm.
- There are config files for lots of other apps.
